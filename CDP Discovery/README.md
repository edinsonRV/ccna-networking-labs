\# CDP Discovery Lab



\## Objective

Configure and verify Cisco Discovery Protocol (CDP) to discover directly connected Cisco devices and identify neighboring devices.



\## Topics Covered

\- CDP configuration

\- Neighbor discovery

\- Interface verification

\- Basic network troubleshooting



\## Technologies Used

\- Cisco Packet Tracer

\- Cisco IOS

\- Switching and Routing



\## Configuration



\### Router Configuration

```bash

enable

configure terminal

cdp run

end

show cdp neighbors

show cdp neighbors detail

```



\### Switch Configuration

```bash

enable

configure terminal

cdp run

end

show cdp neighbors

```



\## Verification Commands

```bash

show cdp neighbors

show cdp neighbors detail

show cdp interface

```



\## Skills Practiced

\- Cisco IOS navigation

\- Device discovery

\- Infrastructure verification

\- Basic troubleshooting
