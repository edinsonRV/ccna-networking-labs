\# Static Routing Lab



\## Objective

Configure static routes to enable communication between different networks.



\## Topics Covered

\- Static route configuration

\- Network connectivity

\- Routing table verification

\- Next-hop configuration

\- Basic routing concepts



\## Technologies Used

\- Cisco Packet Tracer

\- Cisco IOS

\- IPv4 Routing



\## Configuration



\### Router Configuration

```bash

enable

configure terminal



ip route DESTINATION\_NETWORK SUBNET\_MASK NEXT\_HOP\_IP



end

```



\## Verification Commands

```bash

show ip route

show running-config

ping

traceroute

```
