\# CCNA Networking Labs



Hands-on networking labs focused on routing, switching, VLANs, security, and network troubleshooting using Cisco Packet Tracer.



\## Labs Included

\- SSH Configuration

\- Static Routing

\- VLAN Configuration

\- Inter-VLAN Routing

\- CDP Discovery

\- Port Security



\## Technologies Used

\- Cisco Packet Tracer

\- Routing and Switching

\- VLANs

\- SSH

\- Static Routing

\- Port Security

\- Trunking



\## Objectives

\- Strengthen networking fundamentals

\- Practice Cisco device configuration

\- Improve troubleshooting skills

\- Build practical CCNA-level experience



\## Repository Structure

Each folder contains:

\- Packet Tracer lab file

\- Lab documentation

\- Configuration examples
